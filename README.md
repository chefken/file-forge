Forge your files. Convert anything. Stay offline.

FileForge is a fast, minimal, and powerful offline file conversion app built with Flutter.
It is designed for speed, privacy, and simplicity — with no reliance on internet services.

---

## Overview

FileForge allows you to:

* Convert images to PDF (multi-page support)
* Crop, rotate, and enhance images before conversion
* Open and view PDF files within the app
* Reorder pages prior to export
* Rename files before saving

---

## Why FileForge

Most file conversion tools:

* depend on internet connectivity
* introduce privacy concerns
* include unnecessary complexity

FileForge focuses on:

* complete offline functionality
* fast local processing
* a clean and minimal interface
